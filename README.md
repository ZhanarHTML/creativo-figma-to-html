# Creativo – Лендинг креативного агентства

Адаптивный одностраничный сайт креативного агентства Creativo, полностью свёрстанный по макету из Figma с использованием Tailwind CSS.

## Технологии

- HTML5  
- Tailwind CSS (через CDN)  
- Google Fonts (Inter)  
- Vanilla JavaScript (для аккордеона)

## Особенности проекта

- Полностью адаптивный дизайн (mobile-first)  
- Pixel-perfect соответствие макету Figma  
- Интерактивный аккордеон с анимацией поворота стрелок  
- Чистый семантический HTML  
- Современные эффекты (hover, shadow, transition)
