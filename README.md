# Creativo – Creative Agency Landing Page

Адаптивный лендинг креативного агентства, свёрстанный по макету из Figma.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Открыть%20сайт-brightgreen?style=for-the-badge&logo=vercel)](https://zhanarhtml.github.io/creativo-figma-to-html/)
[![Figma](https://img.shields.io/badge/Figma-Макет-blue?style=for-the-badge&logo=figma)](https://www.figma.com/design/SB9vY91gcHo5WU38wuuL6B/Creative-Agency-Landing-Page--FREE-SAMPLE---Community-?node-id=1-1157&m=dev)

## Технологии
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript (аккордеон)
- Google Fonts (Inter)

## Особенности
- Pixel-perfect по макету
- Полностью адаптивный (mobile-first)
- Интерактивный аккордеон с поворотом стрелок
- Чистый семантический код
